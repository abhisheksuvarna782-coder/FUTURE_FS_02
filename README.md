## Mini CRM Dashboard

A modern, production-ready Mini CRM application built as part of the **Future Interns – Task 2** assignment

## 🚀 Live Demo
🔗 Frontend: https://transcendent-cranachan-682e46.netlify.app/

🔗 Backend API: https://future-fs-02-vqv0.onrender.com

---

## 📌 Features

- Add customers with validation
- View customers in real-time
- Delete customers instantly
- Live dashboard analytics
- Donut chart (customer distribution)
- Line chart (customer growth)
- Professional dark UI / UX
- Fully responsive design

---

## 🛠 Tech Stack.

### Frontend
- React (Create React App)
- Functional Components & Hooks
- Plain CSS (No UI libraries)
- Chart.js

### Backend
- Node.js
- Express.js
- MongoDB Atlas

---

## 🔐 Backend APIs (Tested via Thunder Client)

- GET `/api/customers`
- POST `/api/customers`
- DELETE `/api/customers/:id`

Backend is deployed and verified.
